== Contributing

Couchbase welcomes community contributions to the Scala SDK.
The https://github.com/couchbase/couchbase-scala-client[Scala SDK source code^] is available on GitHub.
Please see the https://github.com/couchbase/couchbase-scala-client/blob/master/CONTRIBUTING.md[CONTRIBUTING^] file for further information.
